Este es un párrafo de ejemplo en una estructura XHTML 1.0 Strict.

Reglas clave de XHTML:

  • Todo elemento debe estar **cerrado** (ej: `

    `).
  • Las etiquetas vacías deben cerrarse con una barra (ej: `
    ` o ``).
  • Todos los nombres de elementos y atributos deben estar en **minúsculas**.
  • Todos los valores de atributos deben estar **entrecomillados**.
